Файлы хранятся на сервере в виде папок. Клиенты могут взаимодейсововать с сервером через сокет: закачивать и скачивать файлы .
Файлы разбиваются на блоки, а блоки передаются потоком байтов, поскульку это оптимальней чем преобразовывать встроку из-за увеличения размера данных.

Для больших файлов - прихождящие блоки файла, дописывать в файл на диск по мере поступления блоков, чтобы не было перерасхода оперативной памяти.

Для пересылки комманд можно указывать в начале каждого блока байт определяющий тип блока (служебная информация или файл).
